    • The map method

    In Ruby, the Enumerable module gives the objects of a class a variety of collection-related behaviors.

    Let's have a better look at the map method!

    No matter the enumerable it starts with, map (also callable by collect) will always return an array filled with whatever gets returned by the block each time it runs.

    The returned array is always the same size as the original enumerable. It's elements consist of the accumulated result of calling the code block on each element in the original object in turn. If you want map to modify an array, you can use map!

    .

    The return value of map: map returns a new object -> a mapping of the original object to a new object.
