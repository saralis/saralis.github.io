    • What is an array? What is a hash?

    Hashes and arrays both store data. In an array, the slots are lined up in a row, and each one is numbered (starting from zero). Ex:

    Syntax

    # Array literal notation
    variable = [item1, item2, ..., itemN]

    Example

    seasons = ['winter', 'spring', 'summer', 'fall']

    In a hash, the slots are not in a row (they are just thrown in together), and you can use any object to refer to a slot, not just a number. Ex:

    Syntax

    # Hash literal notation

    hash_ex = {
    key1: value1,
    key2: value2
    }

    Example

    hash_pets = {
    name: "Bob",
    age: 91
    }
    => { :name => "Artur", :age => 21}

    When do we use arrays and hashes?

    Arrays are best used when you are working with an ordered list of items. Every slot in the list acts like a variable: you can see what object a particular slot points to, and you can make it point to a different object. You can make an array by using square brackets [] In Ruby, As mentioned earlier, the first value in an array has index 0.

    It is good to use hashes when you have a bunch of things you want to keep track of but they do note really fit into an ordered list. You can make an array by using brackets {}.
