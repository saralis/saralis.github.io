The position Property

The position property specifies the type of positioning method used for an element. There are four types of positions:

  • static
  • absolute
  • relative
  • fixed

Let's have a better look at these values!

static

static is the default value an element is set to if a positioning type is not specified.This value simply places the element where it would normally go on the page. If you don't tell an element how to position itself, it just plunks itself down in the document.

relative

relative positioning tells the element to move relative to where it would have landed if it just had the default static positioning. If you give an element relative positioning and tell it to have a margin-top of 20px, it moves down twenty pixels from where it's static positioing.

absolute

absolute positions elements in relation to the first parent element it has that does NOT have position:static. If there's no such element, the element with position:absolute gets positioned relative to html.

fixed

fixed positioning anchors (or "glues") an element to the browser window. If you scroll up and down, the fixed element stays put even as other elements scroll past.
